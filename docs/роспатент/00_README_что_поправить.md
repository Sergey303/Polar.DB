## Что нужно будет поправить / заполнить перед подачей

1. **Объём программы** — указать после подготовки архива/депонируемых материалов.
2. **Год создания версии 2.1.2** — указать точный год завершения/публикации этой версии.
3. **Адрес для переписки** — указать адрес, e-mail и телефон лица/организации, которые будут получать корреспонденцию.
4. **Подписант от правообладателя** — указать ФИО, должность и основание полномочий, если заявка подаётся от института без доверенности.
5. **Реквизиты правообладателя** — проверить ИНН/КПП/ОГРН/адрес по актуальной карточке организации.
6. **Состав депонируемых материалов** — после сборки PDF/A/архива вписать фактические имена файлов, количество листов и размер.
7. **Основание принадлежности исключительного права правообладателю** — внутренне проверить: служебное произведение, договор, приказ, акт передачи или иное основание.
8. **Персональные данные** — заполнить паспортные/адресные/контактные сведения только в официальных формах и согласиях, если они требуются при выбранном способе подачи.

## Рекомендуемый порядок действий

1. Зафиксировать исходники именно версии 2.1.2: NuGet package/tag/commit.
2. Подготовить PDF/A с депонируемыми материалами.
3. Посчитать размер программы/архива.
4. Вписать размер в реферат и карточку заявки.
5. Заполнить официальное заявление Роспатента/ФИПС по данным из файла `01_карточка_заявки_для_формы_ФИПС.md`.
6. Приложить титульный лист, депонируемые материалы, реферат, согласия и подтверждение оплаты пошлины.
