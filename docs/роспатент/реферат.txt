Программа для ЭВМ «Polar.DB» предназначена для встраиваемого хранения и обработки структурированных данных в приложениях на платформе .NET. Программа обеспечивает описание схем данных через типы Polar, запись и чтение последовательностей, доступ к полям записей по именам, построение и использование индексов для поиска по ключу, а также восстановление служебного состояния хранилища после повторного открытия файлов. Область применения — программные системы, которым требуется локальное файловое хранение, индексный доступ, последовательная дозапись и контроль согласованности данных, индексов и состояния. Функциональные возможности: сериализация типизированных данных, работа с последовательностями фиксированной и переменной длины, построение индексов, поиск первого совпадения по ключу и recovery/refresh-обработка граничных состояний.
