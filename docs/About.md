# О проекте

Замысел проекта сформировался в результате неудовлетворенности попытками сконструировать хорошую
базу для RDF-данных. В течение 2012 года были созданы экспериментальные варианты RDF-движков
средствами MS SQL Server, MongoDB и Cassandra. Главными проблемами реализаций были невысокая
производительность и, главное, неспособность загружать и обрабатывать данные среднего размера — 50
млн. триплетов и более. Для NoSQL-решений нередко добавлялась и нестабильность в работе. В первой
половине 2013 года была осуществлена попытка «вручную», через API работы с файлами, сконструировать
RDF-хранилище. Попытка оказалась успешной: удалось загрузить базу данных, содержащую более 1 млрд.
триплетов (`freebase`), в компьютер со скромной оперативной памятью 16 Гб и выполнить на ней
некоторые виды запросов.

<a id="about-goal"></a>
## Цель проекта

Цель проекта — создать библиотеку работы со структурированной информацией, эффективную для
построения специализированных и универсальных баз данных и систем управления базами данных. Кроме
библиотеки классов, в проекте нарабатываются целостные решения по конкретным
задачам.
