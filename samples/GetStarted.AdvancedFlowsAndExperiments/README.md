# GetStarted.AdvancedFlowsAndExperiments

Этот архив собирает оставшиеся более экспериментальные сценарии из старых `GetStarted*` проектов.

Что внутри:
- новый `Program.cs`, который запускает сценарии по id;
- старые partial-`Program` файлы сохранены почти без изменений;
- сценарии-обёртки лежат в папке `Scenarios/`;
- абсолютные пути заменены на локальную папку `data/AdvancedFlowsAndExperiments/`.

Важное замечание:
- часть сценариев по-прежнему тяжёлая по объёму данных и времени выполнения;
- `Main304SQLite` требует `System.Data.SQLite.Core`;
- цель этого архива — вынести advanced/experimental код в один тематический проект, не потеряв исходные комментарии и структуру сценариев.
