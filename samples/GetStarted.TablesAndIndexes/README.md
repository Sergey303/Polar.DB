# GetStarted.TablesAndIndexes

Тематический проект для сценариев про таблицы и индексы.

Что сюда вошло:
- `samples/GetStarted1/Program2.cs`
- `samples/GetStarted1/Program3.cs`

Что здесь важно:
- каждый старый сценарий вынесен в отдельный `.cs` файл;
- новый `Program.cs` только выбирает и запускает сценарий;
- комментарии из исходных файлов сохранены рядом с кодом;
- абсолютный `dbpath` заменён на локальную папку `data/TablesAndIndexes/`.

Команды:
- `dotnet run -- list`
- `dotnet run -- gs1-p2`
- `dotnet run -- gs1-p3`
- `dotnet run -- all`

Замечание:
- в исходных файлах были очень большие объёмы данных; для учебного и локального запуска размеры уменьшены, но логика сценариев сохранена.
