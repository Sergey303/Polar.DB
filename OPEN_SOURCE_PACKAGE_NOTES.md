# Polar.DB open-source community files package

This archive contains a ready-to-apply first batch of community-health and repository-entry files for Polar.DB.

## What is included

Canonical GitHub-recognized files:

- `README.md`
- `LICENSE`
- `CONTRIBUTING.md`
- `CODE_OF_CONDUCT.md`
- `SECURITY.md`
- `SUPPORT.md`
- `.github/ISSUE_TEMPLATE/*`
- `.github/PULL_REQUEST_TEMPLATE.md`
- `CODEOWNERS`
- `CITATION.cff`

Helpful translated copies for Russian-speaking contributors:

- `README.ru.md`
- `CONTRIBUTING.ru.md`
- `CODE_OF_CONDUCT.ru.md`
- `SECURITY.ru.md`
- `SUPPORT.ru.md`
- `.github/ISSUE_TEMPLATE/bug_report.ru.yml`
- `.github/ISSUE_TEMPLATE/feature_request.ru.yml`
- `.github/PULL_REQUEST_TEMPLATE.ru.md`

## Recommended next manual edits

Before committing, review and adjust:

1. contacts for private security reporting, if you want something more explicit than GitHub private reporting;
2. the exact repository owner(s) in `CODEOWNERS`;
3. the author list in `CITATION.cff`, if you want to add or remove names;
4. package metadata in `src/Polar.DB/Polar.DB.csproj`;
5. any historical links in older docs and project metadata.

## Practical note

GitHub natively recognizes the canonical English filenames. The Russian copies are included as optional translated companions for contributors and maintainers.
