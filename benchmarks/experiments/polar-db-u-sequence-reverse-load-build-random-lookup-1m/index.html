Polar.DB USequence exact reference load + build + random GetByKey, 1M persons

Experiment: polar-db-u-sequence-reverse-load-build-random-lookup-1m

Exact benchmark variant of uploaded Program (1).cs: creates USequence, loads 1,000,000 person records in reverse id order, calls Build(), performs GetByKey(npersons*2/3), then performs 10,000 Random().Next(npersons) GetByKey calls. No reopen/Refresh is performed in this experiment.
